钱币规则
钱币
什么是钱币？

钱币是面对小皇叔用户推出的一种增值服务，数量或时间等的一种统计代码，并非任何代币票券，不能用于小皇叔增值服务以外的任何商品或服务。

如何获取钱币？

1.注册即送1000钱币。

2.每介绍1个注册会员积1分。

3.每分享1次信息积0.5分。

4.坚持每天连续签到，每天均可获得1个钱币奖励，连续签到7天以上，每天可获得5个钱币。

5.下单购买商城商品，根据用户的等级不同，返回不同比例的钱币给会员。按照消费金额返钱币。

6.下单所返钱币在订单确认收货后15天返回到会员帐户中，若订单发生退款，则退款部分不记录钱币返比。

7.每个小皇叔账号每天仅能签到一次，对于任何通过不正当手段参与活动者，小皇叔有权取消其签到资格与领取奖励资格。

8.用户所得钱币，不会在年底清除。钱币不支持商品购买时使用。

如何使用钱币？

钱币可在钱币商城兑换相应商品，具体兑换内容，请去钱币商城查看
