推广帮助

我们的用意

小皇叔为了回馈您的热心支持，精心打造了“我的推广”这个模块，希望 您能在享受并且肯定我们蚕丝被优质服务的时候，也能分享给更多人。

推广流程：

第一步：您可以分享您的二维码或者分享我们商城的链接给其他人。

第二步：用户扫描了您分享的二维码或者通过您分享的链接，注册成为我们小皇叔的新用户时，恭喜您，您就成为了这个新用户的邀请人。

第三步：该用户在我们小皇叔顺利消费。再次恭喜您，您将会获得一定的佣金。
佣金金额 = 该新用户有效金额 X 佣金比例。

温馨提示：

1 商品订单佣金生效的条件是该订单没有发生退货，订单顺利完成。

2 服务订单佣金生效的条件是该订单（检验未通过的订单，退差额的订单除外）顺利完成， 佣金按照订单第一次支付的价格计算，补差额的订单，补的金额属于第二次支付，这部分钱将不计入有效金额当中。

3 每月20号系统会自动发放您的所有佣金到您的微信账户中。当然您必须绑定微信号才能正常 完成提现哦。
